مفتاح Gemini API (مجاني)

عند مشاركة الرابط على GitHub: كل مستخدم يُدخل مفتاحه المجاني مرة واحدة. المفتاح يُحفظ فقط في متصفّحه ولا يُرفع مع الملف.

  1. افتح aistudio.google.com/apikey
  2. أنشئ مفتاح API مجاني (Create API key)
  3. الصق المفتاح واضغط حفظ — سيُخفى فوراً ولن يظهر للآخرين
